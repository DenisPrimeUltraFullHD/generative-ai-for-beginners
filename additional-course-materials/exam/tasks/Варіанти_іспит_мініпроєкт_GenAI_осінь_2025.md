## 30 індивідуальних варіантів
Виконуйте у секції [4]. Оберіть завдання за номером вашого варіанту.

**Стандартна структура кожного варіанту**: генеруйте 3+ приклади → валідуйте формат → порівняйте результати → зробіть висновки.

### Блок A: Основи комп'ютерних систем (1-10)
1. **Відеокарти**: Створіть JSON для 3 GPU (NVIDIA, AMD, Intel) `{brand, vram, cuda_cores, price}`. Додайте коротке порівняння «ціна/можливості».
2. **Накопичувачі**: Опишіть 3 типи (HDD, SSD, USB flash) у JSON `{type, capacity, speed, durability}`. Порівняйте придатність для різних задач.
3. **Операційні системи**: Згенеруйте 3 ОС (Windows, Linux, macOS) у JSON `{name, interface, hardware_support, use_cases}`. Порівняйте 2–3 сценарії використання.
4. **Пам'ять (RAM)**: Опишіть 3 типи пам'яті (DDR3, DDR4, DDR5) у JSON `{type, speed, capacity, compatibility}`. Додайте висновок про апгрейд.
5. **Мережеві пристрої**: Описати 3 пристрої (router, switch, access point) у JSON `{device, function, speed, range}`. Дайте рекомендацію для дому/офісу.
6. **Файлові системи**: Створіть JSON для 3 ФС (NTFS, ext4, APFS) `{name, features, max_file_size, compatibility}`. Додайте типовий кейс.
7. **Монітори**: Створіть JSON для 3 типів (LCD, OLED, CRT) `{type, resolution, refresh_rate, power_consumption}`. Порівняйте енергоефективність.
8. **Процесори**: Згенеруйте 3 CPU (Intel, AMD, ARM) у JSON `{brand, cores, clock_speed, power_usage}`. Порівняйте продуктивність/енергію.
9. **Комп'ютерні компоненти**: Опишіть 3 компоненти (CPU, RAM, SSD) у JSON `{name, function, speed, cost}`. Порівняйте «вартість/користь».
10. **Принтери та сканери**: Згенеруйте 3 пристрої (laser, inkjet, 3D printer) у JSON `{type, technology, cost_per_page, quality}`. Додайте короткий висновок.

### Блок B: Програмне забезпечення та мережі (11-20)
11. **Хмарні сервіси**: Створіть JSON для 3 сервісів (Google Drive, Dropbox, OneDrive) `{service, free_space, price, sync_speed}`. Порівняйте обмеження.
12. **Месенджери**: Згенеруйте 3 месенджери (Telegram, WhatsApp, Signal) у JSON `{name, encryption, file_size_limit, users}`. Поясніть відмінності приватності.
13. **Веб-браузери**: Згенеруйте 3 браузери (Chrome, Firefox, Safari) у JSON `{name, engine, speed, extensions}`. Додайте по 1 сильній/слабкій стороні.
14. **Бази даних**: Згенеруйте 3 типи БД (SQLite, MySQL, MongoDB) у JSON `{type, scalability, query_language, use_cases}`. Порівняйте під WEB-проєкт.
15. **Офісні пакети**: Опишіть 3 пакети (MS Office, LibreOffice, Google Docs) у JSON `{suite, features, collaboration, cost}`. Порівняйте колаборацію.
16. **Антивіруси**: Створіть JSON для 3 антивірусів (Kaspersky, Windows Defender, Avast) `{name, detection_rate, system_impact, price}`. Додайте застереження.
17. **Відеоконференції**: Опишіть 3 платформи (Zoom, Teams, Google Meet) у JSON `{platform, max_participants, quality, price}`. Порівняйте для навчання.
18. **Контейнери**: Створіть JSON для 3 технологій (Docker, Podman, LXC) `{name, ease_of_use, performance, security}`. Порівняйте «простота/безпека».
19. **Віртуалізація**: Опишіть 3 рішення (VMware, VirtualBox, Hyper-V) у JSON `{software, performance, features, cost}`. Додайте рекомендацію.
20. **Файлові обмеження (мережа/обмін)**: Згенеруйте 3 способи передачі файлів (email, cloud link, messenger) у JSON `{method, max_file_size, speed, privacy}`.

### Блок C: Інтернет речей та embedded системи (21-30)
21. **Бездротові модулі**: Згенеруйте 3 модулі (WiFi, Bluetooth, LoRa) у JSON `{module, range, power_consumption, price}`. Порівняйте для IoT-сенсора.
22. **Сенсори**: Створіть JSON для 3 сенсорів (temperature, humidity, motion) `{sensor, accuracy, range, price}`. Додайте типове застосування.
23. **Протоколи передачі даних**: Згенеруйте 3 протоколи (I2C, SPI, UART) у JSON `{protocol, speed, pins, complexity}`. Порівняйте складність інтеграції.
24. **Мікроконтролери**: Згенеруйте 3 MCU (Arduino, ESP32, Raspberry Pi Pico) у JSON `{name, pins, programming_language, cost}`. Порівняйте для навчального проєкту.
25. **IoT платформи**: Опишіть 3 платформи (Arduino IoT, AWS IoT, Azure IoT) у JSON `{platform, protocol, cost, complexity}`. Додайте висновок.
26. **Актуатори**: Створіть JSON для 3 виконавчих механізмів (servo, stepper, DC motor) `{type, precision, power, cost}`. Порівняйте точність.
27. **Джерела живлення**: Створіть JSON для 3 джерел (battery, solar, USB) `{type, capacity, voltage, cost}`. Порівняйте автономність.
28. **Екрани та дисплеї**: Опишіть 3 типи (LCD, OLED, e-ink) у JSON `{type, power_consumption, color, cost}`. Порівняйте читабельність.
29. **Розробка плат**: Опишіть 3 варіанти (breadboard, PCB, prototype board) у JSON `{type, reusability, cost, complexity}`. Додайте рекомендацію.
30. **Корпуси та механіка**: Згенеруйте 3 матеріали (plastic, metal, 3D printed) у JSON `{material, durability, cost, manufacturing}`. Порівняйте швидкість виготовлення.
